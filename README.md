# 契約更新リスト自動生成ツール

## 概要

バックオフィス業務における契約更新管理をPythonで自動化したツールです。  
入居者CSVデータを読み込み、更新日・案内日を自動計算してExcelレポートとして出力します。

手作業でのExcel管理では更新漏れや転記ミスが発生しやすいという課題を、  
Pythonによる自動化で解決することを目的として作成しました。

---

## 使用技術

- Python 3.14
- pandas
- openpyxl

---

## 機能

- CSVファイルから入居者データを読み込み
- 契約開始日から2年後の**更新日**を自動計算
- 更新日の3ヶ月前の**案内日**を自動計算
- 更新月別の件数を集計
- Excelファイルに2シート構成で出力（一覧・集計）
- ヘッダーの書式設定・列幅の自動調整

## 使い方

### 1. 必要なライブラリをインストール

```bash
pip install pandas openpyxl
```

### 2. CSVファイルを用意

`tenants.csv` を以下の形式で作成し、同フォルダに置く。

```
tenant_id,tenant_name,room_number,contract_start,rent
1,田中○郎,101,2022-04-01,80000
```

### 3. スクリプトを順番に実行

```bash
python step1_load.py       # データ確認
python step2_datetime.py   # 日付計算
python step3_groupby.py    # 集計
python step4_excel.py      # Excel出力
python step5_format.py     # 書式設定・案内日追加（最終出力）
```

### 4. 出力ファイルを確認

`契約更新リスト.xlsx` が生成されます。

## 制作背景

バックオフィス業務において、契約更新管理はExcelの手作業で行われることが多く、
更新漏れや案内タイミングのずれが業務リスクになるケースがあります。

このツールはその課題をPythonで解決するために作成しました。
RPA導入支援の経験を活かし、**繰り返し作業の自動化**という観点で設計しています。
